#Brain-Sync: 뇌과학 기반 능동적 학습 AI 튜터
강의 자료를 분석하여 사용자에게 정답 대신 역질문을 던지고, 스스로 답을 찾게 유도하여 장기 기억 형성을 돕는 학습 지원 시스템입니다.

1. 개발 배경
기존 AI 학습 도구들은 질문에 즉각적인 해답을 제공하여 사용자의 사고 과정을 단축시킵니다. 이는 지식 습득 효율을 저하시키는 원인이 됩니다. 본 프로젝트는 학습자가 정보를 직접 인출할 때 기억이 강화된다는 원리를 시스템적으로 구현하여 학습 효율을 극대화하고자 합니다.

2. 핵심 구현 로직
단순한 질답 챗봇이 아닌, 아래의 인지 과학 원리를 LLM API 프롬프트에 적용하여 구현했습니다.

Active Recall: 사용자가 읽은 내용을 기억에서 강제로 인출하도록 핵심 개념 위주의 역질문을 생성합니다.

Error-Driven Learning: 사용자가 오답을 입력할 경우 정답을 바로 노출하지 않고, 논리적 모순을 지적하는 힌트를 제공하여 스스로 수정하게 만듭니다.

Metacognition: 단계별 비계 설정을 통해 학습자가 본인의 지식 공백을 객관적으로 인지하도록 유도합니다.

Data Pipeline: PDF/PPT 강의 자료를 파싱하여 텍스트를 추출하고 Vector DB에 저장하여 RAG 시스템을 구축합니다.

Tutor Logic: Claude 4.6 API를 활용하여 사용자의 답변을 분석하고, 뇌과학 로직에 따른 다음 질문이나 힌트를 실시간으로 생성합니다.

Tech Stack: FastAPI, Next.js, Claude 4.6 API, PostgreSQL, Pinecone, Docker

4. 주요 기능
강의 자료 업로드 및 지식 구조화: 업로드된 파일 내 핵심 개념 및 연관 관계 분석

Socratic Interaction: 소크라테스식 발문법을 통한 단계별 힌트 및 유도 질문 제공

학습 분석 리포트: 세션 종료 후 오답 패턴 분석 및 취약 개념 요약 제공
