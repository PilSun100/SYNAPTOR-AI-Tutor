# Demo Scenario

이 문서는 과제 발표 또는 시연에서 사용할 기본 흐름입니다.

## Scenario

1. React 웹 앱을 엽니다.
2. 사용자가 PDF 학습 자료를 업로드합니다.
3. 서버가 파일 타입과 크기를 검증합니다.
4. 서버가 PDF에서 텍스트를 추출합니다.
5. 사용자가 추출된 자료를 기반으로 핵심 개념 추출을 실행합니다.
6. AI가 개념, 설명, 난이도를 반환합니다.
7. 사용자가 학습 세션을 시작합니다.
8. AI가 개념별 Active Recall 질문을 제시합니다.
9. 사용자가 텍스트 답변을 입력합니다.
10. AI가 답변의 정확도, 누락 개념, 오개념 여부를 평가합니다.
11. 사용자가 막히면 Level 1부터 Level 5까지 힌트를 요청합니다.
12. 정답에 도달하면 사용자가 자기 설명을 제출합니다.
13. 세션 종료 후 리포트에서 스스로 맞힌 개념, 힌트 후 맞힌 개념, 반복 오답 개념, 다음 복습 추천을 확인합니다.

## Presentation Points

- 이 프로젝트는 단순 요약 서비스가 아니라 능동적 회상 기반 AI 튜터입니다.
- 정답을 바로 알려주지 않고 예측 오류와 단계적 힌트를 통해 학습자가 스스로 도달하게 합니다.
- 웹 UI와 API 서버를 함께 제공하므로 시연성과 구조 설명이 모두 가능합니다.
- 기능별 Git branch와 commit으로 개발 과정을 확인할 수 있습니다.
